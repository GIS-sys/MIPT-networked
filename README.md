# How to install

1) clone the repository, cd into it

2)
```bash
git submodule update --init --recursive
git submodule sync
git submodule update --recursive --remote
```

# How to run

1)
```bash
./compile.sh
```

2) Launch server and client

```bash
./build/w1/server
```

```bash
./build/w1/client
```
